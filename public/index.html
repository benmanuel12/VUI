How to use

To search one file Select the file with the file selector Required regex /(please )?(do (a )?)?(search|look) for (([\w|\s])+)/ or /(please )?find (\w+)( in (\w+))?/ If using second regex, omit the optional ( in (\w+))? at the end to ensure single file search

To search multiple files Select the files with the file selector Required regex /(please )?(do (a )?)?(search|look) for (([\w|\s])+)/ or /(please )?find (\w+)( in (\w+))?/

All files must follow the format below # Title then the file must have either regular text ### or headings this is all the code will pick up
